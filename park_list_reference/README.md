# Reference for Park List used for the New York State Park Ticks Tracker Database

This directory includes the CSV file that was filtered and pushed to the NYS Park Quality Tracker Database. This file was referenced in order to make a table of park names and locations that could be read from the Database. The file (State_Park_Facility_Points_12_2024.csv) was developed by the [New York State Office of Parks, Recreation and Historic Preservation (2024)](https://data.ny.gov/Recreation/State-Park-Facility-Points/9uuk-x7vh/about_data). 

This data can be downloaded from  the "State Park Facility Points" database, linked below: 
https://data.ny.gov/Recreation/State-Park-Facility-Points/9uuk-x7vh/about_data
